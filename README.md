# Projeto Integrador I - Frontend

## Stack

This project uses:

- [React](https://react.dev/)
- [TypeScript](https://www.typescriptlang.org/docs/)
- [Next.js Using App Router](https://nextjs.org/docs)
- [Material-Ui](https://mui.com/material-ui/getting-started/)

## Running locally

To run in dev mode:

```shell
npm run dev
```
